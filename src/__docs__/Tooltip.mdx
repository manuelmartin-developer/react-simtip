# Tooltip

A tooltip component.

#### Example

<Canvas of={Tooltip.Default} />

## Usage

```ts
import {Tooltip} from "react-simtip";

const Example = () => {
  return (
      <Tooltip
        content="Tooltip content"
      >
        <span>Hover me</span>
      </Tooltip>
  );
};

export default Example;
```

#### Arguments

- **children** - ReactNode - The content to be wrapped by the tooltip.
- **content** - string | ReactNode - The content of the tooltip.
- **placement** - "top" | "bottom" | "left" | "right" - The placement of the tooltip.
- **className** - string - The class name to be applied to the tooltip.
- **showDelay** - number - The delay in milliseconds before the tooltip is shown.
- **disableInteractive** - boolean - Whether the tooltip should be interactive.
  offset - string - The offset of the tooltip.
- **hasArrow** - boolean - Whether the tooltip should have an arrow.
- **animation** - "fade" | "scale" | "flip" | "slide" | "swing" | "bounce" "flicker" - The animation of the tooltip.
- **animationDuration** - number - The duration of the animation in milliseconds.
- **backgroundColor** - string - The background color of the tooltip.
- **color** - string - The text color of the tooltip.
- **padding** - string - The padding of the tooltip.
- **borderRadius** - string - The border radius of the tooltip.
- **fontSize** - string - The font size of the tooltip.
